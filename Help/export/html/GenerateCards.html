Descrição do Plugin

Esta ferramenta permite a exportação de Cartas Topográficas em formato PDF e GeoTIFF, com base em configurações estabelecidas em arquivos JSON e layouts definidos em arquivos QPT.

Seleção do Tipo de Produto

Na caixa Selecione o tipo de produto, o usuário poderá escolher dentre cinco opções:

  • - Carta Topográfica
  • - Carta Ortoimagem
  • - Carta Ortoimagem OM
  • - Carta Ortoimagem Militar
  • - Carta Topográfica Militar

Configuração dos Arquivos JSON

Para realizar a exportação, o usuário deve selecionar um ou mais arquivos JSON de configuração que contenham as especificações do produto a ser exportado. Esses arquivos definem os parâmetros de escala, simbologia, camadas a serem exportadas, e informações de layout.

Para saber mais sobre como configurar um arquivo JSON, e como realizar o download do arquivo clique no botão abaixo:

Ajuda para Configurar Arquivo JSON

Parâmetros de Conexão

No campo Parâmetros de conexão, o usuário deve fornecer as informações necessárias para que o plugin possa se conectar aos bancos de dados geoespaciais referenciados no arquivo JSON. Isso garante que as camadas de dados sejam carregadas corretamente durante a exportação.

Configuração do Layout (QPT)

O layout de impressão do produto é configurado em um arquivo de projeto no formato QPT. Esse layout define elementos importantes como margens, escalas, rótulos, logotipos e créditos, que serão exibidos no produto final exportado.

Para garantir que o layout seja adequado, é importante configurar o arquivo creditsDSG no QPT, que permite alterar os créditos do produto, nome do projeto, logotipo, e outros elementos visuais. A correta configuração deste arquivo garante que o produto atenda às normas e padrões de apresentação.

Para saber mais sobre como configurar o layout de projeto (QPT), clique no botão abaixo:

Ajuda para Configurar Arquivo de Projeto (QPT)
