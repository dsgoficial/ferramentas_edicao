Configuração do Arquivo de Projeto (QPT)

O arquivo de projeto (QPT) é utilizado para configurar o layout dos produtos cartográficos. Ele define elementos como créditos, logos e informações adicionais que aparecem nas cartas exportadas. Caso esteja desenvolvendo um novo projeto ou precise alterar detalhes como a logo ou os créditos, você pode editar o arquivo QPT.

Alterando o Arquivo QPT (Créditos)

O arquivo padrão utilizado pelo plugin é o creditsDSG.qpt. Se não for necessário customizar o layout dos créditos, este arquivo será carregado automaticamente. No entanto, se houver necessidade de alterar elementos como a logo ou outros detalhes, será necessário editar esse arquivo ou criar um novo arquivo QPT.

Quando Pode Ser Necessário Alterar o Arquivo

Talvez seja necessário alterar o creditsDSG.qpt caso o projeto tenha algum convênio onde seja preciso representar a logo e o nome da instituição parceira, ou se precisar mudar o nome do projeto ou sua descrição. Isso só precisa ser feito uma vez e, se for alterado, é necessário apontar o novo arquivo no JSON; caso contrário, será utilizado o arquivo padrão.

Importante

Não é recomendável que o operador altere o QPT dos produtos sem necessidade, principalmente em Cartas Ortoimagem, onde a legenda é dinâmica. Alterações inadequadas podem desconfigurar a exportação e comprometer o uso do plugin. Além disso, o tamanho do creditsDSG.qpt impacta no posicionamento de outros elementos do rodapé, portanto, mantenha o padrão sempre que possível.
