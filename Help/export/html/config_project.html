Configuração do Arquivo de Projeto (QPT)

O arquivo de projeto (QPT) é utilizado para configurar o layout dos produtos cartográficos. Ele define elementos como créditos, logos e informações adicionais que aparecem nas cartas exportadas. Caso esteja desenvolvendo um novo projeto ou precise alterar detalhes como a logo ou os créditos, você deve editar o arquivo QPT.

Alterando o Arquivo QPT (Créditos)

O arquivo padrão utilizado pelo plugin é o creditsDSG.qpt. Se não for necessário customizar o layout dos créditos, este arquivo será carregado automaticamente. No entanto, se houver necessidade de alterar elementos como a logo ou outros detalhes, será necessário editar esse arquivo ou criar um novo arquivo QPT.


Exemplo: Modelo Padrão

OBSERVAÇÃO: Se houver necessidade de configurar um arquivo específico, o usuário deve especificar o novo caminho no arquivo JSON.

Abrir a Pasta de Configuração

Abrir Pasta de Configuração com os Layouts (QPT)
