Orientações Iniciais

Este plugin foi desenvolvido para funcionar apenas em inglês na versão do 3.24-Tisler. Caso seja necessário mude o idioma do QGIS em Configurações > Opções > Geral e depois reinicie o QGIS
Para garantir a exibição correta das fontes, e o funcionamento de algumas ferramentas é necessário instalar as fontes Noto Sans disponíveis para instalação abaixo.

Instalar Fontes Noto Sans

As ferramentas disponíveis podem ser visualizadas na imagem a seguir:

Aba de Ferramentas de Edição

Note que, para algumas dessas ferramentas (Sentido de Corrente, Fluxo de Vegetação e Suprimir Bandeira), é necessário alterar o tipo de produto (Topográfica ou Ortoimagem), e/ou o seletor de escala (1:25.000, 1:50.000, 1:100.000 ou 1:250.000).

Importante: Para garantir o funcionamento correto (WYSIWYG), certifique-se de que o banco de dados esteja sempre projetado no sistema de referência SIRGAS2000 UTM com o fuso correto. Isso ajudará a evitar problemas de distância e a garantir o posicionamento preciso dos rótulos.
