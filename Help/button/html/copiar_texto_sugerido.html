Copiar Texto Sugerido

• IMPORTANTE: Para ativar a ferramenta "Rótulo Lago", clique no ícone referente a ferramenta, conforme o exemplo acima, lembre-se também de selecionar o produto correto que você está trabalhando (Carta Topográfica ou Ortoimagem) e de selecionar a escala da carta.

Descrição:

A ferramenta "Copiar Texto Sugerido" permite copiar rótulos sugeridos automaticamente pelo QGIS para a camada edicao_texto_generico_l. Essa ferramenta usa os dados gerados automaticamente durante a rotulagem e cria feições de rótulo baseadas no estilo e nas configurações de texto do rótulo sugerido.

Modo de Uso:

  1. Selecione a camada que contém as feições com rótulos a serem copiados.
  2. Clique no botão "Copiar Texto Sugerido" na barra de ferramentas de edição.
  3. Selecione a feição que deseja copiar o rótulo. A ferramenta copiará suas configurações, como fonte, tamanho e cor.
  4. Essas informações serão usadas para criar uma nova feição na camada edicao_texto_generico_l, dependendo da geometria do rótulo.

Dicas para Configuração:

  • Verifique se a camada ativa possui o campo texto_edicao configurado corretamente.
  • Essa ferramenta é útil em projetos de mapeamento que envolvem uma grande quantidade de rótulos similares, como rodovias ou trechos de drenagem.

Aplicação Prática:

Em casos onde rótulos sugeridos automaticamente precisam ser replicados para camadas específicas mantendo o estilo, a ferramenta Copiar Texto Sugerido facilita o processo, copiando automaticamente os rótulos gerados e suas configurações.

Exemplo: Copiando rótulos sugeridos automaticamente para a camada edicao_texto_generico_l.

Limitações:

A ferramenta Copiar Texto Sugerido só funcionará corretamente em camadas que possuem o campo texto_edicao configurado. Caso o campo esteja ausente ou mal configurado, a ferramenta exibirá uma mensagem de erro.
