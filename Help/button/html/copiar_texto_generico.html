Copiar Texto Genérico

• IMPORTANTE: Para ativar a ferramenta "Rótulo Lago", clique no ícone referente a ferramenta, conforme o exemplo acima, lembre-se também de selecionar o produto correto que você está trabalhando (Carta Topográfica ou Ortoimagem) e de selecionar a escala da carta.

Descrição:

A ferramenta "Copiar Texto Genérico" permite copiar feições selecionadas de uma camada ativa e transferi-las para as camadas edicao_texto_generico_p ou edicao_texto_generico_l. O texto é transferido junto com a geometria da feição, facilitando a inserção automática de rótulos.

Esta ferramenta é ideal para criar rótulos rapidamente com base em feições existentes, sem a necessidade de criar rótulos manualmente ou ajustar o texto posteriormente.

Modo de Uso:

  1. Selecione as feições desejadas na camada ativa que contenha o atributo nome (o valor do atributo não pode ser nulo).
  2. Clique no botão "Copiar Texto Genérico" na barra de ferramentas de edição.
  3. A ferramenta verificará se o atributo nome está presente na camada. Se o atributo não existir, uma mensagem de erro será exibida.
  4. Para feições do tipo ponto, a geometria será copiada para a camada edicao_texto_generico_p, e para feições do tipo linha, será copiada para a camada edicao_texto_generico_l.
  5. O valor do campo nome será atribuído ao campo texto_edicao na camada de destino, e o estilo padrão será aplicado ao texto.

Dicas para Configuração:

  • Certifique-se de que o campo nome está presente e preenchido nas feições antes de usar a ferramenta.
  • Utilize a ferramenta para criar rótulos genéricos rapidamente.

Aplicação Prática:

Situações onde feições como pontos de interesse ou linhas de drenagem, por exemplo, precisam ser rotuladas rapidamente, a ferramenta Copiar Texto Genérico permite que você copie as feições para camadas específicas e adicione rótulos automaticamente, agilizando o processo de rotulagem.

Exemplo: Copiando feições e inserindo rótulos automáticos com base no atributo "nome".

Limitações:

A ferramenta Copiar Texto Genérico só funcionará corretamente em camadas que possuem o campo nome não nulo. Caso contrário, a ferramenta não poderá copiar o texto e exibirá uma mensagem de erro.
