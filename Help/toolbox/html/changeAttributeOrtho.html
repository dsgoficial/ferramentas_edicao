Help versão 1.0

Descrição:

O algoritmo atualiza os atributos das camadas selecionadas para configurar os atributos de edição de produtos cartográficos do tipo Ortoimagem. Ele permite alterar os atributos de diversas feições de acordo com a tabela de características, incluindo elementos naturais, infraestruturas e áreas específicas, como elementos hidrológicos, pistas de pouso e massas d'água.

Parâmetros:

• Selecione as camadas a serem alteradas → Escolher as camadas vetoriais (de qualquer geometria) cujos atributos serão configurados.
• Selecione a escala de edição → Indicar a escala do produto.

Exemplo de uso:

Configuração dos atributos de camadas de hidrografia e infraestrutura:

• Selecione as camadas a serem alteradas → elemnat_elemento_hidrografico_a, infra_elemento_energia_p
• Selecione a escala de edição → 1:50.000

Aplicação:

Feições que foram expostas devido à atribuição "sim" na coluna "visível" por ação da rotina.
