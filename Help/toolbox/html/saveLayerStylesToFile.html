Help versão 1.0

Descrição:

O algoritmo exporta estilos de camadas dentro de um grupo selecionado e os salva como arquivos QML na pasta de destino especificada.

Parâmetros:

• Selecione o grupo de camadas → Definir o grupo cujos estilos serão exportados.
• Selecione a pasta de destino → Definir a pasta onde os arquivos QML serão salvos.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

Exemplo de parâmetros usados:

• Selecione o grupo de camadas → bloco4_2794-3-SE_2024_edicao
• Selecione a pasta de destino → caminho na máquina de trabalho.
