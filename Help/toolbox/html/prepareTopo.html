Help versão 1.0

Descrição:

Este algotitmo tem por objetivo preparar o conjunto de dados para a fase de edição.

Parâmetros:

• Camadas de entrada → Indicar todas as camadas para serem configuradas para a edição. • Selecione a escala de edição → Indicar a escala da Carta Topográfica. • Selecionar camada de moldura → Indicar a camada com a moldura do projeto. Atenção: Esta rotina deve ser utilizada exclusivamente durante o preparo para a edição. Sua aplicação em qualquer outro momento pode resultar na perda de trabalho, pois ela ajusta automaticamente diversos atributos. Portanto, é essencial seguir este procedimento apenas na fase de preparação, garantindo que os ajustes necessários sejam realizados de forma segura e apropriada.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Camadas de entrada → Todas as camadas do CDGV • Selecione a escala de edição → 1:25.000 • Selecionar camada de moldura → aux_moldura_a

Aplicação:

Atributos expostos devido a ação da rotina.
