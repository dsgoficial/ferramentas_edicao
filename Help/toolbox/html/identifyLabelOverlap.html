Help versão 1.0

Descrição:

O algoritmo identifica sobreposições entre os rótulos das feições, para que o operador os mova a fim de possibilitar uma melhor visualização na leitura do produto cartográfico.

Parâmetros:

• Selecionar camadas: → Indicar as camadas que serão utilizadas como entrada no processo. • Limite geográfico [optional] → Indicar a moldura do projeto. • Escala → Indicar a escala do produto cartográfico. • Flags Sobreposição de Rótulos → Camada resultado do processo. Por padrão gera uma camada temporária. Caixa de seleção permite carregar a camada resultado no canvas Qgis.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar camadas: → Todas as camadas • Limite geográfico [optional] → aux_moldura_a • Escala → 1:25.000 • Flags Sobreposição de Rótulos → [Create temporary layer]

Aplicação:

Sobreposições de rótulos identificadas.
