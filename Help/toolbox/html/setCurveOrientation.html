Help versão 1.0

Descrição:

O algoritmo organiza a orientação dos textos de cotas das curvas de nível de acordo com o parâmetro escolhido.

Parâmetros:

• Selecionar camada de curva de nível → Indicar a camada que contém as curvas de nível. • Selected features only → Há possibilidade de rodar o processo apenas na(s) feição(ões) selecionada(s). • Orientação → Sentido de orientação das curvas de nível.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar camada de curva de nível → elemnat_curva_nivel_l • Selected features only → Desmarcado. • Orientação → Sentido horário.

Aplicação:

Curvas de nível orientadas conforme o parâmetro selecionado.
