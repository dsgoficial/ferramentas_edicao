Descrição:

O algoritmo reprojeta coordenadas armazenadas nos atributos de uma camada de acordo com os sistemas de coordenadas de referência (SRC).

Parâmetros:

• Input Vector Layers → Indicar as camadas que serão utilizadas na rotina. • Label X → Indicar o atributo com a coordenada X. • Label Y → Indicar o atributo com a coordenada Y. • Source EPSG → Indicar o Sistema de referência atual. • Destination EPSG → Indicar o Sistema de referência futuro.

Exemplo de uso:

• Input Vector Layers → constr_edificacao_p • Label X → label_x • Label Y → label_y • Source EPSG → 4674 • Destination EPSG → 31983
