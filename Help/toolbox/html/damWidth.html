Descrição:

O algoritmo define a largura de representação das barragens levando em consideração, inclusive, as sobreposições com a camada de vias de deslocamento.

Parâmetros:

• Selecionar camada de barragem → Indicar a camada que contém as barragens com geometria tipo linha. • Selecionar o atributo de largura → Indicar o atributo da camada que contém a informação de largura. • Selecionar camada de rodovia → Indicar a camada que contém as vias de deslocamento.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar camada de barragem → infra_barragem_l • Selecionar o atributo de largura → largura_simbologia • Selecionar camada de rodovia → infra_via_deslocamento_l

Aplicação:

Atributo 'largura_simbologia' preenchido adequadamente.
