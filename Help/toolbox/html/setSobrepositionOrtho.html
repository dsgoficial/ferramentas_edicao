Help versão 1.0

Descrição:

O algoritmo configura a sobreposição de linhas de áreas públicas militares, terras indígenas e unidades de conservação, permitindo verificar e ajustar a interseção dessas camadas com drenagem, vias de deslocamento e ferrovias.

Parâmetros:

• Selecione a camada de edição de área pública militar linha → Escolher a camada vetorial de área pública militar para verificar a sobreposição.
• Selecione a camada de edição terra indígena linha → Escolher a camada vetorial de terra indígena para verificar a sobreposição.
• Selecione a camada de unidade de conservação linha → Escolher a camada vetorial de unidade de conservação para verificar a sobreposição.
• Selecione a escala de edição → Indicar a escala do produto.

Exemplo de uso:

Verificação de sobreposição de áreas protegidas com vias de deslocamento e drenagem:

• Selecione as camadas de drenagem, vias de deslocamento e ferrovia → elemnat_trecho_drenagem_l, infra_via_deslocamento_l, infra_ferrovia_l
• Selecione as áreas públicas militares, terras indígenas e unidades de conservação → llp_area_pub_militar_a, llp_terra_indigena_a, llp_unidade_conservacao_a

Aplicação:

Feições de sobreposição configuradas com base nas áreas de interesse e interseções encontradas nas camadas de drenagem e infraestrutura.
