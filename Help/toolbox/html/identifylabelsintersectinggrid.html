Help versão 1.0

Descrição:

O algoritmo identifica rótulos que intersectam com as linhas do grid da carta, sinalizando textos mal posicionados que prejudicam a legibilidade do produto cartográfico e devem ser reposicionados pelo operador.

Parâmetros:

• Selecionar camadas: → Indicar as camadas que serão utilizadas como entrada no processo. • Selecionar a Camada do Grid: → Indicar a camada multilinestring que contém o grid da carta. • Escala: → Indicar a escala do produto cartográfico. • Flags Rótulos Intersectando Grid: → Camada resultado do processo. Por padrão gera uma camada temporária. Caixa de seleção permite carregar a camada resultado no canvas Qgis.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar camadas: → Todas as camadas com rótulos • Selecionar a Camada do Grid: → aux_grid_carta_l • Escala: → 1:25.000 • Flags Rótulos Intersectando Grid: → [Create temporary layer]

Aplicação:

Rótulos que intersectam o grid da carta são identificados e sinalizados para reposicionamento.
