Help versão 1.0

Descrição:

O algoritmo mescla linhas contíguas com base no ângulo de deflexão entre elas, criando uma única linha contínua.

Parâmetros:

• Selecione a camada de linhas.
• Definir o número máximo de iterações.

Exemplo de uso:

Mesclagem de linhas em uma camada de rodovias.

Exemplo de parâmetros usados:

• Parâmetros aplicados com valores específicos para execução da ferramenta.

Aplicação:

Imagem representativa da ferramenta em ação.
