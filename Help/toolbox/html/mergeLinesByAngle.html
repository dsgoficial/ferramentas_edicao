Descrição:

O algoritmo mescla linhas contíguas com base no ângulo de deflexão entre elas, criando uma única linha contínua.

Parâmetros:

• Input layer → Escolher a camada que terá as feições mergeadas. • Iteration limit → Limite de iterações. • Merged Lines → Resultado do processo. Por padrão gera uma camada temporária, mas pode salvar em um destino selecionado.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Input layer → elemnat_trecho_drenagem_l • Iteration limit → 5 • Merged Lines → Criar camada temporária
