Help versão 1.0

Descrição:

O algoritmo mescla os trechos de drenagem presentes na camada vetorial, criando uma representação contínua e simplificada da hidrografia dentro dos limites geográficos fornecidos. A rotina gera uma camada temporária com os rios mergeados, o operador deverá inspecionar o resultado a fim de identificar possíveis erros.

Parâmetros:

• Selecione a camada de drenagem → Escolher a camada que contém os trechos de drenagem ou rios a serem mesclados. • Selecione a camada de moldura → Escolher a camada que define os limites geográficos nos quais os rios serão mesclados.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecione a camada de drenagem → elemnat_trecho_drenagem_l • Selecione a moldura geográfica → aux_moldura_a

Aplicação:

Resultado com os vetores mergeados.
