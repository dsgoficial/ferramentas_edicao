Help versão 1.0

Descrição:

O algoritmo mescla segmentos de rios ou trechos de drenagem presentes na camada vetorial, criando uma representação contínua e simplificada da hidrografia dentro dos limites geográficos fornecidos.

Parâmetros:

• Selecione a camada de drenagem → Escolher a camada que contém os trechos de drenagem ou rios a serem mesclados.
• Selecione a camada de moldura → Escolher a camada que define os limites geográficos nos quais os rios serão mesclados.

Exemplo de uso:

Mesclagem de trechos de drenagem dentro de uma moldura geográfica:

• Selecione a camada de drenagem → elemnat_trecho_drenagem_l
• Selecione a moldura geográfica → aux_moldura_a

Aplicação:

Segmentos de rios foram mesclados para criar uma representação contínua da hidrografia dentro da área especificada.
