Descrição:

O algoritmo define a rotação das pontes e bueiros de acordo com os relacionamentos das feições com as camadas de rodovia, trecho de drenagem e vala.

Parâmetros:

• Selecionar camada de elemento viário → Indicar a camada que contém os elementos viários com geometria do tipo ponto. Há possibilidade de executar apenas para as feições selecionadas. • Selecionar o atributo de rotação → Indicar o atributo da camada acima que contém a informação da rotação da simbologia. • Tolerância da distância → Indicar a distância a ser considerada no buffer em relação às camadas dos relecionamentos. Atenção: A depender do sistema de referência utilizado, será necessário a indicação da tolerância em graus, para um sistema geográfico, ou em metros, no caso de um sistema projetado. • Selecionar camada de rodovia → Indicar a camada que contém as vias de deslocamento. • Selecionar camada de trecho de drenagem → Indicar a camada que contém os trechos de drenagem. • Selecionar camada de vala → Indicar a camada que contém as valas.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar camada de elemento viário → infra_elemento_viario_p • Selecionar o atributo de rotação → simb_rot • Tolerância da distância → 0,00001 • Selecionar camada de rodovia → infra_via_deslocamento_l • Selecionar camada de trecho de drenagem → elemnat_trecho_drenagem_l • Selecionar camada de vala → infra_vala_l

Aplicação:

Atributo 'simb_rot' preenchido adequadamente.
