Help versão 1.0

Descrição:

O algoritmo generaliza os pontos cotados para gerar uma representação mais simplificada no diagrama de elevação.

Parâmetros:

• Selecione a camada contendo os pontos cotados → Indicar a camada que contém as feições de ponto cotado. • Selecione o campo que informa a cota → Indicar o atributo que contém a informação sobre a cota. • Limite geográfico → Indicar a moldura do projeto. • Selecione a escala para gerar as quadrículas do diagrama de elevação → Indicar a escala do produto cartográfico. • Camadas geradas → Resultados da rotina, por padrão gera camadas temporárias.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecione a camada contendo os pontos cotados → elemnat_ponto_cotado_p • Selecione o campo que informa a cota → cota • Limite geográfico → aux_moldura_a • Selecione a escala para gerar as quadrículas do diagrama de elevação → 1:25.000 • Camadas geradas → Criar camadas temporárias

Aplicação:

Imagem representativa da ferramenta em ação.
