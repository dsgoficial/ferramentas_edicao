Help versão 1.0

Descrição:

O algoritmo insere na camada de edição as simbologias previstas na ET-RDG para cada edificação com geometria do tipo área, considerando a rotação adequada.

Parâmetros:

• Selecionar camada de edificação área → Indicar a camada que contém as edificações com geometria do tipo área. • Selecionar camada de edição → Indicar a camada que contém as simbologias de representação dos tipos de cada edificação. • Selecionar o atributo de rotação da camada de edição → Indicar o atributo que contém a informação de rotação das simbologias.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar camada de edificação área → constr_edificacao_a • Selecionar camada de edição → edicao_simb_area_p • Selecionar o atributo de rotação da camada de edição → simb_rot

Aplicação:

Simbologia inserida e rotacionada adequadamente.
