Help versão 1.0

Descrição:

O algoritmo generaliza os pontos cotados com base em critérios definidos e na escala de edição, assim, omite alguns pontos a fim de melhorar a visualização das informações.

Parâmetros:

• Selecione a escala para gerar as quadrículas: → Indicar a escala do produto cartográfico. • Selecione a camada contendo os pontos cotados → Indicar a camada que contém as feições de ponto cotado. • Selecione a camada contendo as curvas de nível → Indicar a camada que contém as curvas de nível. • Limite Geográfico → Indicar a camada que contém a moldura do projeto. • Selecione o campo que informa se é depressão ou não → Indicar o atributo da curva de nível que contém a informação sobre depressão. • Selecione o campo que informa se o ponto é visível ou não → Indicar o atributo do ponto cotado que contém a informação sobre visibilidade. • Generalização de Ponto Cotado → Camada resultado do processo.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecione a escala para gerar as quadrículas: → 1:25.000 • Selecione a camada contendo os pontos cotados → elemnat_ponto_cotado_p • Selecione a camada contendo as curvas de nível → elemnat_curva_nivel_l • Limite Geográfico → aux_moldura_a • Selecione o campo que informa se é depressão ou não → depressao • Selecione o campo que informa se o ponto é visível ou não → visivel • Generalização de Ponto Cotado → Criar camada temporária

Aplicação:

Vetor de grade gerado e reclassificado em "edicao_grid_edicao_l".
