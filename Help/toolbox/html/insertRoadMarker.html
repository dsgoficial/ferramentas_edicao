Help versão 1.0

Descrição:

O algoritmo insere os identificadores de trecho rodoviário utilizando as informações dos atributos da camada infra_via_deslocamento_l.

Parâmetros:

• Selecionar camada de via de deslocamento → Indicar a camada que contém as vias de deslocamento. • Selecionar camada de identificador trecho rodoviário → Indicar a camada de edição que contém os pontos identificadores de trecho rodoviário. • Selecionar a camada de moldura → Indicar a camada que contém a moldura do projeto. • Selecionar a escala de edição → Indicar a escala do produto cartográfico.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar camada de via de deslocamento → infra_via_deslocamento_l • Selecionar camada de identificador trecho rodoviário → edicao_identificador_trecho_rod_p • Selecionar a camada de moldura → aux_moldura_a • Selecionar a escala de edição → 1:25.000

Aplicação:

Identificadores de trecho rodoviário Estadual e Federal, respectivamente, inseridos adequadamente.
