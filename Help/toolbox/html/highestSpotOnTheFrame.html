Help versão 1.0

Descrição:

O algoritmo permite definir a cota mais alta dentro da área delimitada por uma moldura. Para isso realiza a alteração no atributo de cota mais alta da camada de ponto cotado.

Parâmetros:

• Selecionar camada de ponto cotado → Indicar a camada que contém as feições de ponto cotado. • Selecionar o atributo de "cota" → Indicar o atributo de ponto cotado que contém a informação de cota. • Selecionar o atributo de "cota mais alta" → Indicar o atributo de ponto cotado que contém a informação de cota mais alta. • Selecionar camada de moldura → Indicar a camada que contém a moldura do projeto.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar camada de ponto cotado → elemnat_ponto_cotado_p • Selecionar o atributo de "cota" → cota • Selecionar o atributo de "cota mais alta" → cota_mais_alta • Selecionar camada de moldura → aux_moldura_a

Aplicação:

Cota mais alta definida pelo processo.
