Help versão 1.0

Descrição:

O algoritmo gera um diagrama de elevação a partir de uma camada de pontos cotados.

Parâmetros:

• Selecione a camada de pontos cotados.
• Selecione a extensão geográfica.
• Selecione a escala de edição.

Exemplo de uso:

Geração de diagrama de elevação para uma área delimitada.

Exemplo de parâmetros usados:

• Parâmetros aplicados com valores específicos para execução da ferramenta.

Aplicação:

Imagem representativa da ferramenta em ação.
