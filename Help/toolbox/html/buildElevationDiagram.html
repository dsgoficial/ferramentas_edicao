Help versão 1.0

Descrição:

O algoritmo gera um diagrama de elevação a partir do modelo digital de elevação.

Parâmetros:

• Input DEM → Indicar o raster do modelo digital de elevação. • Equidistance value → Indicar a equidistância das curvas de nível do produto cartográfico. • Geographic bounds layer → Indicar a moldura do produto cartográfico. • Area without information layer → Indicar a camada de área sem informação (Opcional). • Water bodies layer → Indicar a camada que contém as feições de corpo d'água.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Input DEM → MDT_MI_2794-3-SE • Equidistance value → 10 • Geographic bounds layer → aux_moldura_a • Area without information layer → Não indicado • Water bodies layer → cobter_massa_dagua_a

Aplicação:

Diagrama de elevação gerado pela rotina.
