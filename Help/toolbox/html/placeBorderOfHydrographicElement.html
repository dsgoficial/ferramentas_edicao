Help versão 1.0

Descrição:

O algoritmo insere bordas em feições hidrográficas, como massas d'água e cursos fluviais, criando uma representação mais detalhada das fronteiras desses elementos no mapa.

Parâmetros:

• Selecione a camada de elemento hidrografico → Escolher a camada que contém as feições hidrográficas a serem editadas.
• Executar somente nas feições selecionadas → Determinar se a ação será realizada apenas nas feições atualmente selecionadas.
• Selecione a camada de massa d'água → Escolher a camada que contém as massas d'água relacionadas.
• Selecione a camada de borda de elemento hidrográfico → Escolher a camada onde as bordas dos elementos hidrográficos serão inseridas.

Exemplo de uso:

Inserção de bordas em massas d'água selecionadas:

• Selecione a camada de elemento hidrografico → elemnat_elemento_hidrografico_a
• Selecione a camada de massa d'água → cobter_massa_dagua_a
• Selecione a camada de borda de elemento hidrográfico → edicao_borda_elemento_hidrografico_l

Aplicação:

Bordas foram inseridas ao redor dos elementos hidrográficos selecionados, como massas d'água e rios.
