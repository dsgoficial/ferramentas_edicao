Help versão 1.0

Descrição:

O algoritmo salva as configurações de máscaras aplicadas nas camadas de rótulos e exporta as informações para um arquivo .json.

Parâmetros:

• Selecionar o arquivo para salvar → Definir um caminho para salvar o arquivo .json

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar o arquivo para salvar → "C:/caminho/mascaras.json"

Atenção: Ao dar um nome pro arquivo lembrar de colocar a extensão ".json"
