Help versão 1.0

Descrição:

O algoritmo ordena as camadas e aplica os estilos para a edição.

Parâmetros:

• Tipo da carta → Indicar o produto e linha de produção do SAP. • Selecione a escala de edição → Indicar a escala do produto. • Grupo → Indicar o CDGV do produto. • Deseja exibir curvas auxiliares → Permite a visualização de curvas de nível auxiliares. • Definir equidistância fora do padrão → Equidistância diferente do previsto para a escala do trabalho. Indicação em metros.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Tipo da carta → Carta Topográfica 1.3 • Selecione a escala de edição → 1:25.000 • Grupo → bloco4_2794-3-SE_2024_pos_reamb • Deseja exibir curvas auxiliares → Não • Definir equidistância fora do padrão → Not set

Aplicação:

Camadas ordenadas e com os estilos de edição carregados.
