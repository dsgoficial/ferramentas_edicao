Help versão 1.0

Descrição:

A rotina altera o atributo 'sobreposto' da camada de limite legal, caso essas camadas tenham interseção com uma camada a ser sobreposta. As entradas são do tipo linha e o atributo a ser alterado é o "sobreposto" da camada de limite legal.

Parâmetros:

• Selecionar a camada de moldura → Indicar a camada que contém a(s) moldura(s) do projeto. • Selecionar a camada de edicao_limite_legal → Indicar a camada que contém o limite legal. • Selecione a camada de trecho de drenagem → Indicar a camada que contém os trechos de drenagem. • Selecione a camada de via de deslocamento → Indicar a camada que contém as vias de deslocamento. • Selecione a camada de ferrovia → Indicar a camada que contém as ferrovias.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar a camada de moldura → aux_moldura_a • Selecionar a camada de edicao_limite_legal → llp_limite_legal_l • Selecione a camada de trecho de drenagem → elemnat_trecho_drenagem_l • Selecione a camada de via de deslocamento → infra_via_deslocamento_l • Selecione a camada de ferrovia → infra_ferrovia_l

Aplicação:

Sobreposição de limites corrigida.
