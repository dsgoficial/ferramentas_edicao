Help versão 1.0

Descrição:

O algoritmo insere na camada de edição as simbologias previstas na RDG para cada feição de extração mineral com geometria do tipo área.

Parâmetros:

• Selecionar camada de extração mineral → Indicar a camada que contém as feições de extração mineral com geometria do tipo área. Permite executar a rotina somente nas feições selecionadas. • Selecionar o atributo de "visibilidade" da camada de entrada → Indicar o atributo da camada de entrada que contém a informação sobre a visibilidade. • Ocultar feições que não tem tamanho suficiente → Oculta feições com tamanho insuficiente para serem representadas como área. • Selecione a escala de edição → Indicar a escala do produto cartográfico. • Selecionar camada de edição → Selecionar a camada que contém as simbologias de representação.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar camada de extração mineral → constr_extracao_mineral_a • Selecionar o atributo de "visibilidade" da camada de entrada → visivel • Ocultar feições que não tem tamanho suficiente → Desmarcado • Selecione a escala de edição → 1:25.000 • Selecionar camada de edição → edicao_simb_area_p

Aplicação:

Simbologia de extração mineral inserida adequadamente.
