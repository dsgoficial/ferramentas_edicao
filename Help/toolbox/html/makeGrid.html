Help versão 1.0

Descrição:

O algoritmo gera o vetor de grade em UTM no fuso correto baseado na moldura e reprojeta para o sistema de referência de interesse

Parâmetros:

• Selecione a camada de moldura → Indicar a camada com a moldura do projeto. Há possibilidade de rodar o processo apenas na(s) moldura(s) selecionada(s). • Selecione a escala de edição → Indicar a escala do produto. • Gerar cruzetas de lat long → Gera cruzetas de encontro da latitude com a longitude de acordo com as marcações das margens. • Vetor de grade → Por padrão, cria uma camada temporária contendo o vetor gerado. Há possibilidade de indicar um caminho para salvar o vetor resultante do processo. • Open output file after running algotithm → Carregar a camada resultante no Canvas do QGIS. Atenção: Após gerar o vetor temporário, verificar se está corretamente construído e reclassificar para a camada correta: "edicao_grid_edicao_l".

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecione a camada de moldura → aux_moldura_a • Selecione a escala de edição → 1:25.000 • Gerar cruzetas de lat long → Marcado • Vetor de grade → Padrão - Criar camada temporária. • Open output file after running algotithm → Marcado

Aplicação:

Vetor de grade gerado e reclassificado em "edicao_grid_edicao_l".
