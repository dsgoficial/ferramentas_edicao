Help versão 1.0

Descrição:

O algoritmo insere a simbologia prevista na RDG para cada tipo de vegetação a depender da escala do produto cartográfico. Atenção: Tipos de vegetação com simbologia prevista: Reflorestamento (1296); Caatinga (801); Camparina (501); Cerrado (701); Restinga (401); Terreno exposto - pedregoso (1003).

Parâmetros:

• Selecionar camada de vegetação → Indicar a camada com as feições de vegetação. Há possibilidade de rodar o processo apenas nas feições selecionadas. • Selecione a escala de edição → Indicar a escala do produto. • Selecionar camada de edição de símbolo de vegetação → Indicar a camada de edição que contém as feições com a simbologia prevista para as vegetações.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar camada de vegetação → cobter_vegetacao_a • Selecione a escala de edição → 1:25.000 • Selecionar camada de edição de símbolo de vegetação → edicao_simb_vegetacao_p

Aplicação:

Símbolo de vegetação, Reflorestamento (1296), inserido adequadamente.
