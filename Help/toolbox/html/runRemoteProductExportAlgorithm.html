Help versão 1.0

Descrição:

O algoritmo permite executar a exportação de produtos remotamente, utilizando uma interface JSON.

Parâmetros:

• Endereço do serviço de exportação.
• JSON de exportação.
• Exportar TIFF.

Exemplo de uso:

Exportação de um produto de Carta Topográfica.

Exemplo de parâmetros usados:

• Parâmetros aplicados com valores específicos para execução da ferramenta.

Aplicação:

Imagem representativa da ferramenta em ação.
