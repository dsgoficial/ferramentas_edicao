Help versão 1.0

Descrição:

O algoritmo permite executar a exportação de produtos remotamente, utilizando uma interface JSON.

Parâmetros:

• Endereço do serviço de exportação → Indicar o ip de endereço do banco de dados. • JSON de exportação → Indicar o caminho para o arquivo json de exportação. • texto do json de exportação → Prévia do arquivo json. • tipo da carta → Indicar o tipo de produto cartográfico. • Usuário do banco de dados → Indicar o usuário do banco de dados. • Senha do usuário do banco de dados → Indicar o senha do banco de dados. • Parâmetros de conexão do Proxy → Indicar os parâmetros de conexão proxy • Pasta para salvar os arquivos exportados → Indicar o caminho para salvar o pdf.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):31983

• Endereço do serviço de exportação → localhost • JSON de exportação → "C:/caminho/exportacao.json" • texto do json de exportação → Prévia do arquivo json. • tipo da carta → Carta Topográfica 1.3 • Usuário do banco de dados → poostgres • Senha do usuário do banco de dados → ******** • Parâmetros de conexão do Proxy → Conforme convém • Pasta para salvar os arquivos exportados → "C:/caminho/arquivo.pdf"
