Help versão 1.0

Descrição:

O algoritmo permite carregar e aplicar máscaras predefinidas nas camadas de edição, utilizando um arquivo .json que contém a configuração das máscaras.

Parâmetros:

• Selecione o arquivo .json de máscaras → Indicar o caminho do arquivo .json que contém as informações das máscaras. • Selecione as camadas que terão máscaras aplicadas → Indicar as camadas que terão as máscaras aplicadas.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecione o arquivo .json de máscaras → "C:/caminho/mascaras.json" • Selecione as camadas que terão máscaras aplicadas → Todas as camadas selecionadas.
