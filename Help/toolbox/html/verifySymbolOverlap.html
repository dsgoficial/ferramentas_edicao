Help versão 1.0

Descrição:

O algoritmo verifica se alguma simbologia sobrepõe outra dentre as camadas pre-definidas, gerando uma camada de polígonos contendo as sobreposições, indicando possíveis correções ao operador.

Parâmetros:

• De "Selecionar a camada de via de deslocamento" até "camada de ferrovia" → O prório algoritmo seleciona as camadas de forma automática. • Selecionar a camada de moldura → Indicar a camada que contém a moldura do projeto. • Escala → Indicar a escala do produto cartográfico. • Área mínima → Indicar o tamanho mínimo que será considerado nas interseções entre as simbologias • Flags Sobreposição de Simbologia → Camada de polígonos resultado do processo. Por padrão gera um arquivo temporário.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• De "Selecionar a camada de via de deslocamento" até "camada de ferrovia" → Indicação automática • Selecionar a camada de moldura → aux_moldura_a • Escala → 1:25.000 • Área mínima → 40 • Flags Sobreposição de Simbologia → Padrão - Criar camada temporária

Aplicação:

Polígono indicando sobreposição de símbologias, resultado após a correção manual.
