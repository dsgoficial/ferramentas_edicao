Help versão 1.0

Descrição:

O algoritmo copia feições específicas de rodovias, ferrovias, valas e trechos de drenagem para a camada de elemento viário, realizando o mapeamento de atributos de acordo com regras estabelecidas. As inserções não são commitadas na base de dados, apenas adicionadas à camada de destino.

Parâmetros:

• Selecionar camada de rodovias → Indicar a camada que contém as rodovias (padrão: infra_via_deslocamento_l). Feições com 'tipo_elemento_viario' diferente de 97 e 9999 serão copiadas para a camada de elemento viário. • Selecionar camada de ferrovias → Indicar a camada que contém as ferrovias (padrão: infra_ferrovia_l). Feições com 'tipo_elemento_viario' diferente de 97 e 9999 serão copiadas para a camada de elemento viário. • Selecionar camada de vala → Indicar a camada que contém as valas (padrão: infra_vala_l). Feições com 'em_galeria_bueiro' igual a 1 serão copiadas para a camada de elemento viário. • Selecionar camada de trecho de drenagem → Indicar a camada que contém os trechos de drenagem (padrão: elemnat_trecho_drenagem_l). Feições com 'em_galeria_bueiro' igual a 1 serão copiadas para a camada de elemento viário. • Selecionar camada de elemento viário → Indicar a camada de destino onde as feições serão inseridas (padrão: infra_elemento_viario_l).

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Selecionar camada de rodovias → infra_via_deslocamento_l • Selecionar camada de ferrovias → infra_ferrovia_l • Selecionar camada de vala → infra_vala_l • Selecionar camada de trecho de drenagem → elemnat_trecho_drenagem_l • Selecionar camada de elemento viário → infra_elemento_viario_l

Mapeamento de atributos:

Para Rodovias (infra_via_deslocamento_l):

• tipo_elemento_viario → tipo
• nome_elemento_viario → nome
• posicao_pista_elemento_viario → posicao_pista
• material_construcao_elemento_viario → material_construcao
• modal_uso = 4
• situacao_fisica = 3

Para Ferrovias (infra_ferrovia_l):

• tipo_elemento_viario → tipo
• nome_elemento_viario → nome
• posicao_pista_elemento_viario → posicao_pista
• material_construcao_elemento_viario → material_construcao
• modal_uso = 5
• situacao_fisica = 3

Para Valas e Trechos de Drenagem:

• tipo = 501
• material_construcao = 0
• modal_uso = 4
• situacao_fisica = 3
• posicao_pista = 97

Aplicação:

O algoritmo copia feições com base nos critérios especificados. As feições são adicionadas à camada de elemento viário com os atributos devidamente mapeados, sem realizar commit na base de dados.
