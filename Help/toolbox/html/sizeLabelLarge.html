Help versão 1.0

Descrição:

O algoritmo ajusta o tamanho da fonte dos rótulos em feições de pontos e linhas, utilizando a largura do polígono de referência e a escala de edição. Essa configuração é útil para melhorar a legibilidade dos mapas em diferentes escalas.

Parâmetros:

• Camada com os polígonos selecionados → Escolher a camada que contém as feições de polígonos que servirão de referência para o ajuste. • Camada de texto ponto com as feições selecionadas → Escolher a camada que contém os rótulos de texto associados às feições de ponto. • Camada de texto linha com as feições selecionadas → Escolher a camada que contém os rótulos de texto associados às feições de linha. • Selecionar o atributo de "tamanho_txt" da camada de entrada → Selecionar o atributo da camada de entrada que contém a informação de tamanho do texto. • Selecione a escala de edição → Indicar a escala do produto.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):4674

• Camada com os polígonos selecionados → constr_edificacao_a • Camada de texto ponto com as feições selecionadas → constr_edificacao_p • Camada de texto linha com as feições selecionadas → edicao_texto_generico_l • Selecionar o atributo de "tamanho_txt" da camada de entrada → tamanho_txt • Selecione a escala de edição → 1:25.000

Aplicação:

Tamanho do rótulo ajustado conforme a largura das feições de referência e a escala escolhida.
