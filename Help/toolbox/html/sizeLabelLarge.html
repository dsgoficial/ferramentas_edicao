Help versão 1.0

Descrição:

O algoritmo ajusta o tamanho da fonte dos rótulos em feições de pontos e linhas, utilizando a largura do polígono de referência e a escala de edição. Essa configuração é útil para melhorar a legibilidade dos mapas em diferentes escalas.

Parâmetros:

• Selecione a camada com os polígonos → Escolher a camada que contém as feições poligonais que servirão de referência para o ajuste.
• Selecione a camada de texto ponto → Escolher a camada que contém os rótulos de texto associados às feições de ponto.
• Selecione a camada de texto linha → Escolher a camada que contém os rótulos de texto associados às feições de linha.
• Selecione a escala de edição → Indicar a escala do produto.

Exemplo de uso:

Ajuste do tamanho da fonte em feições de texto genérico:

• Selecione a camada poligonal → aux_area_edicao_a
• Selecione a camada de rótulos de ponto → edicao_texto_generico_p
• Selecione a escala de edição → 1:50.000

Aplicação:

Tamanho do rótulo ajustado conforme a largura das feições de referência e a escala escolhida.
