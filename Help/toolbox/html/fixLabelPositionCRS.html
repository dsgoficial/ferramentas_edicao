Help versão 1.0

Descrição:

O algoritmo corrige as coordenadas de rótulos em camadas, ajustando para o sistema de referência de coordenadas (SRC) correto.

Parâmetros:

• Input Layers → Indicar as camadas que serão utilizadas na rotina. • CRS Antigo → Indicar o Sistema de referência anterior.

Exemplo de uso:

Modelagem EDGV3.0_TOPO na escala de 1:25.000 (EPSG):31983

• Input Layers → constr_edificacao_p • CRS Antigo → 4674

Aplicação:

Coordenadas dos Labels ajustadas conforme ação da rotina.
